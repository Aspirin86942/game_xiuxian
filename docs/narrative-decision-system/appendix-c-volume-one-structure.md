## 1. 目的

本附录用于把第一卷《七玄门风云》的目标结构固定下来，避免“卷纲”只存在于 issue 或聊天里。

本附录描述的是目标态卷结构，不代表当前运行时代码已经完成切换。

## 4. 第一卷跨卷读取点上限

第一卷卷末最多只保留以下三类读取点：

1. `升仙入口`：玩家为何能进入下一卷。
2. `绿瓶风险`：奇物必须继续保密或承担后果。
3. `1 条人物或旧账因果`：例如厉飞雨、墨彩环或墨大夫遗产中的一条主因果。

超出这三类的旧账，原则上都应在第一卷内解释完毕或转换为不可见的低权重背景状态。
